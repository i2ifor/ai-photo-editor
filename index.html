📖 如何获取百度AI密钥

步骤1：访问 百度AI开放平台

步骤2：用手机号注册百度账号并登录

步骤3：进入控制台，点击"创建应用"

步骤4：填写应用信息（随便填就行）

步骤5：创建成功后，你会看到 API Key 和 Secret Key

步骤6：把这两个密钥复制到上面的输入框中
